# WriteRight

AI-powered IELTS Writing evaluation platform.

## Tech Stack
Next.js 14, Supabase, Anthropic Claude, Vercel, PayOS

## Product
- `/evaluate` — AI evaluation with bilingual feedback
- `/dashboard` — history, progress tracking
- `/e/[token]` — shareable full evaluation detail
- `/courses` — LMS course catalog
- `/admin/courses` — admin CRUD
